# Font Files

This directory contains Inter font files for the application.

## Required Font Files:
- Inter-Regular.woff2
- Inter-Bold.woff2  
- Inter-SemiBold.woff2

## How to get these fonts:

### Option 1: Download from Google Fonts
```bash
# Download Inter font family
wget -O inter-fonts.zip "https://fonts.google.com/download?family=Inter"
unzip inter-fonts.zip -d inter-fonts
cp inter-fonts/*.woff2 public/assets/fonts/
